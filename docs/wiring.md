# 接线说明

适用于 **ESP32-S3-DevKitC-1 (N16R8)** + **OV2640** + **HX711** + **MG996R 舵机**。

> 引脚定义与 `firmware/Final/Final.ino` 保持一致。

## 总体原则

1. **所有模块必须共地**（ESP32 GND、HX711 GND、舵机电源负极、摄像头 GND）
2. **舵机必须独立 5V 供电**，不可从 ESP32 开发板 5V 引脚直接带 MG996R
3. 摄像头与 HX711 使用 **3.3V 逻辑**
4. SCCB（I2C）建议 SDA/SCL 各加 **4.7kΩ 上拉** 到 3.3V（部分模组已板载）

---

## OV2640 摄像头

| 模块引脚 | ESP32-S3 引脚 | 说明 |
|----------|---------------|------|
| 3.3V | 3.3V | 供电 |
| GND | GND | 共地 |
| SIO_D (SDA) | GPIO5 | SCCB 数据 |
| SIO_C (SCL) | GPIO4 | SCCB 时钟 |
| VSYNC | GPIO39 | 帧同步 |
| HREF | GPIO40 | 行参考 |
| PCLK | GPIO21 | 像素时钟 |
| D0 | GPIO8 | 数据线 |
| D1 | GPIO9 | 数据线 |
| D2 | GPIO10 | 数据线 |
| D3 | GPIO41 | 数据线 |
| D4 | GPIO11 | 数据线 |
| D5 | GPIO12 | 数据线 |
| D6 | GPIO42 | 数据线 |
| D7 | GPIO47 | 数据线 |
| XCLK | **不接** | 模块自带晶振，无外部时钟引脚 |
| PWDN | 代码中为 -1 | 若模块有 PWDN，可接 GND 或 GPIO 控制 |
| RESET | 代码中为 -1 | 若模块有 RESET，可接 3.3V 或 GPIO 控制 |
| FLASH | **可不接** | 补光灯控制，非成像必需 |

---

## HX711 称重模块

| 模块引脚 | ESP32-S3 引脚 | 说明 |
|----------|---------------|------|
| VCC | 3.3V | 供电 |
| GND | GND | 共地 |
| DT (DOUT) | GPIO1 | 数据输出 |
| SCK | GPIO2 | 时钟输入 |
| E+ / E- / A+ / A- | 称重传感器 | 四线桥式传感器 |

标定系数（当前代码）：`HX711_CALIBRATION_FACTOR = 430.0f`

---

## MG996R 舵机

| 舵机线 | 连接 | 说明 |
|--------|------|------|
| 红线 (VCC) | 外部 5V 正极 | **独立电源** |
| 棕/黑线 (GND) | 外部 5V 负极 + ESP32 GND | **必须共地** |
| 黄/橙线 (Signal) | GPIO18 | PWM 控制 (~50 Hz) |

---

## 推荐接线顺序（调试）

1. **仅 ESP32 + 摄像头** → 验证相机初始化与推理
2. **加 HX711** → 验证重量读数与标定
3. **最后接舵机 + 外部 5V** → 验证 Web 测试与自动喂食

## 电源示意

```text
USB 5V ──► ESP32-S3 (3.3V 逻辑)
                │
                ├── 3.3V ──► OV2640, HX711
                ├── GND  ──┬── 共地
                │          │
External 5V ────┴── GND ──┴── MG996R GND
     │
     └── 5V ───────────────► MG996R VCC

ESP32 GPIO18 ──────────────► MG996R Signal
```

## 已知注意事项

- **GPIO3、GPIO45** 等为 ESP32-S3 启动相关脚，本方案已尽量避开
- 摄像头并口线尽量短，远离舵机大电流线
- 首次上电建议在空载状态下完成 HX711 `tare()`
- 若相机偶发 `decode_err`，代码会自动尝试重新初始化驱动
